# MyBike Landing Page

A responsive landing page for a bicycle brand, showcasing bike models, product comparisons, key features, and a contact form for booking a test ride.

## Live Preview

[View live demo](https://jk-npc.github.io/mybike-landing/)

## Design Reference

[Figma design](https://www.figma.com/design/NZQAIydtHo5QkINyGLHNcq/BIKE-New-Version?node-id=0-1&p=f&t=CcF8AG1OAiKIxLBb-0)

## Technologies Used

- **HTML5** — page structure
- **SCSS** (BEM methodology) — styling
- **JavaScript** — interactivity
- **Gulp** — build automation (SCSS compilation, live reload, image optimization)

## Getting Started

1. Clone the repository:
git clone https://github.com/jk-npc/mybike-landing.git
cd mybike-landing

2. Install dependencies:
npm install

3. Run the project locally:
npm start

## Features

- Responsive design: optimized for different screen sizes, breakpoints at 320px, 744px, and 1260px
- Mobile-friendly navigation menu
- Product comparison section with bike models and pricing
- Detailed feature showcase (Auto Unlock, Range & Lights, Brakes)
- Interactive contact form for booking a test ride
